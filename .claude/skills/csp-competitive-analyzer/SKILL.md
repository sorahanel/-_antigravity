# CSP 경쟁력 분석기

## 기능

1. **CSP 트렌드 분석**: 웹 검색으로 AWS/NCP/Azure/GCP/kt cloud 최신 동향 수집
2. **경쟁 비교 매트릭스**: 다차원 비교표 생성 (가격, 성능, 보안, AI, 지원 등)
3. **윈백 포인트 도출**: 특정 CSP 대비 kt cloud 전환 근거 정리
4. **고객 맞춤 분석**: 고객사 산업/규모/요구사항에 맞는 비교 분석

## 분석 대상 CSP

| CSP | 약칭 | 주요 특징 |
|-----|------|----------|
| Amazon Web Services | AWS | 글로벌 1위, 가장 넓은 서비스 포트폴리오 |
| Naver Cloud Platform | NCP | 국내 2위, CLOVA AI 생태계 |
| Microsoft Azure | Azure | 글로벌 2위, MS 생태계 연동 |
| Google Cloud Platform | GCP | AI/ML 강점, BigQuery |
| kt cloud | KTC | 국내 통신 인프라, 공공/금융 특화 |

## 비교 프레임워크

### 6대 비교 차원

| 차원 | 세부 항목 |
|------|----------|
| **💰 가격/비용** | 컴퓨팅 단가, 네트워크 비용, 약정 할인, 히든 코스트, TCO |
| **🤖 AI/GPU** | GPU 종류(A100/H100/H200), 가용성, AI 플랫폼, MLOps 도구 |
| **🔒 보안/규제** | CSAP 인증 등급, 금융 컴플라이언스, 데이터 주권, 망분리 |
| **⚡ 성능** | 컴퓨팅 벤치마크, 네트워크 레이턴시, SLA 보장 수준 |
| **🛠️ 기술 지원** | 전담 지원 여부, 한국어 지원, 마이그레이션 지원 프로그램 |
| **🌐 생태계** | 파트너 수, 마켓플레이스, 3rd-party 연동, 그룹사 시너지 |

### kt cloud 핵심 차별점 (분석 시 항상 참조)

1. **네트워크**: kt그룹 통신 인프라 기반 국내 최저 레이턴시
2. **공공/금융**: G-Cloud / CSAP 인증, 금융 규제 대응 완비
3. **GPU**: NVIDIA 공식 파트너, A100/H100 GPU 클라우드
4. **데이터 주권**: 국내 데이터센터 운영, 해외 반출 이슈 없음
5. **AI 플랫폼**: kt enterprise AI 통합 플랫폼
6. **가격**: 국내 트래픽 비용 우위, 맞춤형 약정 할인
7. **지원**: 전담 기술 지원팀 + 마이그레이션 전문 인력 배정
8. **그룹 시너지**: kt 통신/미디어/금융 계열사 연계 서비스

## 사용 예제

**입력**: "삼성생명 NCP 윈백용 경쟁 분석해줘"

**처리**:
1. 웹 검색으로 NCP와 kt cloud의 최신 금융권 클라우드 동향 수집
2. 금융 컴플라이언스 관점 중점 비교
3. 삼성생명 규모/요구사항 맞춤 분석

**출력**:
```
⚔️ 삼성생명 NCP→kt cloud 윈백 분석
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 고객 요구사항: 금융권 대기업, 보안/컴플라이언스 최우선, AI 활용 확대 중

| 비교 항목 | kt cloud | NCP | 판정 |
|----------|----------|-----|------|
| 금융 CSAP | ✅ 인증 완료 | ✅ 인증 완료 | ⚖️ 동등 |
| 전용 네트워크 | kt 전용망 직결 | 일반 인터넷 | ✅ kt cloud |
| AI 플랫폼 | enterprise AI | CLOVA | ⚖️ 용도별 |
| 동일 스펙 비용 | ₩XX | ₩XX | ✅ kt cloud |
| 마이그레이션 | 전담팀 배정 | 셀프 가이드 | ✅ kt cloud |

🎯 윈백 핵심 포인트:
1. kt 전용망 연동으로 금융 보안 요건 충족 + 초저지연 보장
2. NCP 대비 동일 스펙 약 15% 비용 절감 가능
3. 전담 마이그레이션 팀 배정으로 전환 리스크 최소화

⚠️ 리스크 및 대응:
- NCP CLOVA 기반 AI 서비스 의존도 → 대체 AI 솔루션 매핑 필요
- 기존 NCP 약정 잔여 기간 / 위약금 → 전환 보상 프로그램 제안
```

**입력**: "최근 CSP 시장 동향 정리해줘"

**처리**: 웹 검색으로 각 CSP의 최근 1~3개월 주요 뉴스/발표 수집 후 구조화
**출력**: CSP 시장 동향 리포트 형식으로 출력
